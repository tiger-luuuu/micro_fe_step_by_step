# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## 项目概述

这是一个微前端技术原理的解构工程，目标是与模型协作，从最基础的内容开始逐步构建一个相对完善的微前端方案，帮助项目负责人完整理解微前端的技术原理。

项目采用渐进式构建方式——每个阶段应能独立理解，再推进到下一阶段。

## 架构目标

本项目旨在探索并实现微前端的核心概念，包括：
- 子应用加载与生命周期管理
- 应用隔离（JS 沙箱、CSS 隔离）
- 应用间通信机制
- 主应用与子应用间的路由协调
- 基于模块联邦（Module Federation）或类似方案的构建层集成

## 随项目演进更新本文档

随着代码库的增长，本文件应持续更新以反映：
- 所选的微前端实现方案（如 single-spa、模块联邦、自定义 iframe 方案等）
- 如何启动主应用和各子应用
- 应用间路由与通信的实现方式
- 各应用的构建与测试命令
