# README.md

这是一个微前端技术原理的解构工程，计划与模型协作，一步一步从最基础的内容开始，构建一个相对完善的微前端方案，帮助项目负责人能完整的理解微前端的技术原理。

| 阶段   | 内容                                            | 状态 |
| ------ | ----------------------------------------------- | ---- |
| 阶段一 | 最小核心：注册表 + 路由调度 + 生命周期 + 脚本加载 | ✅ 完成 |
| 阶段二 | JS 沙箱：Proxy 隔离子应用全局变量               | ✅ 完成 |
| 阶段三 | CSS 隔离：Shadow DOM + Scoped CSS 双方案        | ✅ 完成 |
| 阶段四 | 应用间通信：全局事件总线/共享状态               | 待实现 |
| 阶段五 | 路由协调：主子应用路由联动                      | 待实现 |

## 启动方式

```bash
npx serve packages -p 3000 --cors
```

访问 `http://localhost:3000/main-app/`
